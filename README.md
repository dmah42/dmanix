dmaos
=====
It's an i386 OS.

One time setup:
$ sudo mkdir /mnt/dmaos\_floppy

To build:
$ make
$ update.sh
$ run.sh

Allowed c++ headers:
 <ciso646>
 <cstddef>
 <cfloat>
 <limits>
 <climits>
 <cstdint>
 <cstdlib>
 <new>
 <typeinfo>
 <initializer_list>
 <cstdalign>
 <cstdarg>
 <cstdbool>
 <type_traits>
 <atomic>
